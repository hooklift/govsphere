Fault - CpuIncompatible1ECX

Extends
CpuIncompatible
Since
VI API 2.5

Fault Description

Convenience subclass for calling out some named features among the incompatibilities found in CPUID level 1 register ecx.

Properties

Name Type Description
aesxsd:boolean

Flag to indicate bit 25 is incompatible.

Since vSphere API 5.0
otherxsd:boolean

Flag to indicate that bits other than 0/1/9/19/20/25 are incompatible. I.e. the detected incompatibilities cannot be completely described by the sse3, pclmulqdq, ssse3, sse41, sse42, and/or aes flags.
otherOnlyxsd:boolean

Flag to indicate that the sse3, pclmulqdq, ssse3, sse41, sse42, and aes flags are all false, and the "other" flag is true. Purely a convenience property for the client processing this fault.
