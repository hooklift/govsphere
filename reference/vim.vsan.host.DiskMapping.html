Data Object Description

A VsanHostDiskMapping is a set of one SSD HostScsiDisk backed by a set of one or more non-SSD HostScsiDisk. The maximum allowed VsanHostDiskMapping for a host is 5. A maximum set of 6 non-SSDs HostScsiDisk can be added to the one SSD HostScsiDisk.
See VsanHostConfigInfoStorageInfo
See vim.host.VsanSystem#initializeDisks(DiskMapping[])
