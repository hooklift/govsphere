Data Object Description

This class defines Network Filter Policy.

Properties

Name Type Description
filterConfig*DvsFilterConfig[]

List of Network Filter Configurations. In an update operation, the array can contain all DvsTrafficFilterConfigSpec objects or all DvsFilterConfig and DvsTrafficFilterConfig object, but not mixed of Config and Spec objects. If array of FilterConfigSpec and TrafficFilterConfigSpec is used for updating Network Filter then only the Network Filters matching key / key is updated. If array of DvsFilterConfig and DvsTrafficFilterConfig is used for updating port settings, the Network Filter settings will be overridden with the new array specified. The specified array should only contain FilterConfig and TrafficFilterConfig objects with FilterConfig#inherited / TrafficFilterConfig#inherited set to false. FilterConfig/TrafficFilterConfig objects with FilterConfig#inherited/TrafficFilterConfig#inherited as true in the specified array will be ignored. The updated result will include FilterConfig/TrafficFilterConfig objects inherited from parent, if any.
